# Meteor-Dragula

> Drag and drop so simple it hurts

Browser support includes every sane browser and **IE7+**. <sub>_(Granted you polyfill the functional `Array` methods in ES5)_</sub>

Packaged for Meteor


# Getting Started

```
meteor add ahref:dragula
```
